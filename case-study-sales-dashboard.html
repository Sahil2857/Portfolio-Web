1. Project Overview

Automating End-to-End Sales Reporting

This project automated a multi-file sales reporting workflow, from raw data ingestion to dashboard-ready output. The system standardized datasets, calculated business KPIs, and generated a consistent output for weekly decision-making.

2. Problem

Manual Reporting Was Slow And Error-Prone

  • Multiple Excel files arrived in inconsistent formats.
  • Manual consolidation consumed hours each week.
  • Frequent formula errors reduced confidence in reports.
  • Leadership lacked timely visibility into trends.

Code Preview

Core Automation Logic

automation_pipeline.py

import pandas as pd

raw = pd.concat([pd.read_excel(path) for path in source_files], ignore_index=True)
clean = (
    raw.drop_duplicates()
       .rename(columns=str.strip)
       .fillna({"region": "Unknown"})
)

summary = (clean.groupby("week", as_index=False)
                .agg(revenue=("amount", "sum"),
                     orders=("order_id", "count")))

summary.to_excel("dashboard_output.xlsx", index=False)
